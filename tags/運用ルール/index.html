ブログ運用ルールと初投稿テスト

このブログは、技術的な備忘録や日常のメモ、旅行記などを記録する目的で開設しました。 HugoとPaperModテーマを使って構築しており、今後はコンテンツを投稿しながら運用ルールを整えていく予定です。 この記事では、今後の運用方針や投稿ルールについて記録します。 ■ ブログ運用ルール（初版） ・投稿ディレクトリの命名は blog/日付-タイトル とする（例：blog/2025-07-08-first-post） ・URL構造は /blog/年/月/日/タイトル/ ・カテゴリは tech / travel / blog などを想定 ・画像は記事フォルダ内に格納して同階層から参照 ・OGP画像やSEO設定は今後導入予定 ■ 投稿手順（暫定） hugo new で記事生成 index.md を編集 ローカル確認 → Git commit → push ■ 今後の展望とか ・テンプレート整備 ・OGP/SEO設定 ・Git運用ルールの明文化 ・カテゴリとタグの使い分けルール検討
